Alextra by mazadillon

Web based extension to Alpro herd management

Provides extra features to the Alpro system by DeLaval. Works by accessing the back-end database and copying data into it's own mySQL database. A basic WAMP stack provides a web interface with which to few extra information from Alpro including historic milking & sort gate times and activity information. Also presents a web based milking parlour interface which can be accessed by networked devices such as a parlour console, smart phone over wifi, laptop etc. Imports information from UniformAgri to provide detailed cow information on the parlour display. Provides an interface to allow for transmission of cow IDs to the centre of the parlour (in our case for milk recording).

Install

Will likely need modification to work in an alternative set up, currently designed for a 40 point rotary. Requires setting up the alpro (microsoft access) database in ODBC.
